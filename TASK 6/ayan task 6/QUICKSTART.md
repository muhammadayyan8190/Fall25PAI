# Quick Start Guide 🚀

## اردو میں سمجھنے کے لیے / समझने के लिए

Object Counter ایک ویڈیو تجزیہ ٹول ہے جو خودکار طریقے سے آپ کی ویڈیو میں اشیاء کی شمار کرتا ہے۔

---

## 3 آسان قدم / 3 आसान कदम

### STEP 1: ضروری چیزیں لگائیں / आवश्यक चीजें लगाएं

Double-click کریں: **run.bat** (Windows کے لیے)

یا PowerShell میں یہ کمانڈ چلائیں:
```
python run.py
```

---

### STEP 2: ویب سائٹ کھولیں / वेबसाइट खोलें

اپنے browser میں جائیں:
**http://localhost:5000**

---

### STEP 3: ویڈیو اپ لوڈ کریں / वीडियो अपलोड करें

1. **Upload Video** بکس پر کلک کریں
2. اپنی ویڈیو فائل منتخب کریں
3. **Analyze Video** بٹن دبائیں
4. نتائج دیکھیں!

---

## کیا ہوتا ہے؟ / क्या होता है?

✅ AI (YOLOv8) آپ کی ویڈیو کو دیکھتا ہے  
✅ سب اشیاء کو خودکار طریقے سے شناخت کرتا ہے  
✅ ہر چیز کے ارد گرد سبز باکس بناتا ہے  
✅ کل تعداد دکھاتا ہے  
✅ نئی ویڈیو محفوظ کرتا ہے  

---

## کون سی اشیاء detect کر سکتا ہے؟ / कौन सी चीजें detect कर सकता है?

- **لوگ** / लोग: person
- **گاڑیاں** / गाड़ियाँ: car, bus, truck, bike
- **جانور** / जानवर: dog, cat, bird, horse
- **سامان** / सामान: chair, table, bag, laptop
- اور بہت کچھ...

---

## مسائل حل کریں / समस्याओं को हल करें

**Q: یہ بہت سست ہے / यह बहुत धीमा है**  
A: چھوٹی ویڈیو استعمال کریں (1-2 منٹ)

**Q: Error دستہ ہے / त्रुटि दिखा रहा है**  
A: Python 3.8+ install کریں

**Q: Port 5000 پہلے سے استعمال ہو رہا ہے**  
A: app.py میں `port=5000` کو `port=5001` سے بدل دیں

---

## کیا ہے میے فائلیں؟ / मेरी फाइलें कहां हैं?

📁 **uploads/** = اپ لوڈ کی گئی ویڈیوز  
📁 **static/outputs/** = تیار شدہ ویڈیوز
